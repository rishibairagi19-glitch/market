# QR Based Smart E-Commerce Shop

This is a Python Flask-based e-commerce platform that allows shopkeepers to create their digital stores, add products, manage inventory, and receive orders directly on WhatsApp.

## Features
- 🛒 Auto-generated Public Shop URL & QR Code
- 📦 Product Management (Add/Edit/Hide/Delete)
- ☁️ Supabase Cloud Storage for Images
- 🌗 Dark Mode & Bilingual (Hindi/English) Support
- 📋 Order Management System

## How to Run Locally
1. Clone the repository.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
